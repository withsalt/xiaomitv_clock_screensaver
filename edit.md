参数调节

- 线宽范围
  - 最大线宽： wMax ，默认 8 （ index.html:365 ）
  - 最小线宽： wMin ，默认 1.5 （ index.html:366 ）
- 渐变形状
  - 宽度衰减系数： Math.exp(-3 * x * x) 中的 3 控制收窄速度（ index.html:367 ）
  - 亮度衰减系数： Math.exp(-2.2 * x * x) 中的 2.2 控制中心到两端的亮度过渡（ index.html:368 ）
- 发光强度
  - 阴影模糊： shadowBlur = 4 + 10 * ... 可整体调亮或调暗（ index.html:374 ）
- 平滑度
  - 绘制步进当前为每度 Math.PI / 180 （ index.html:363 ）。如果需要更柔滑，可改为 Math.PI / 360 。
如何进一步微调

- 想让中心更粗、更亮：增大 wMax 和衰减系数中的常数（如把 3 和 2.2 提高一些）。
- 想让两端更细、更淡：减小 wMin 并增大衰减系数。
- 想让弧的位置更靠外或更靠内：调整半径 radius （ index.html:356 ），例如设为 140 或 145 。
